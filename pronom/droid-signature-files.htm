DROID signature files (archive)

  
 

 

This page contains an archive of all previously released signature files. Signature files are generated by PRONOM and used by DROID for file format identification. The signatures are presented here in xml form, each xml file contains multiple signatures that were released at the same time. If you are interested in file format research, or more specifically in file format identification, you may find these files useful. See the accompanying release notes for each signature file.

A note on the numbering of the signature files: PRONOM auto-generates signature files monthly. When no new signature information has been added or we are mid way through an update we do not release the file. So when a gap occurs between two signature files - for instance there is no DROID_SignatureFile_V27.xml, the next file after V26 is V28, this simply means that V27 did not contain significant new data and therefore was not released.
